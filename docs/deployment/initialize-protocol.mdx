# Initialize Protocol

After deployment, all contracts must be initialized:

```bash
npm run initialize
```

## Initialization Steps

### [1/4] USDC Token
```typescript
USDCToken.initialize(
  admin: deployer,    // Can mint test USDC
  name: "USDC",
  symbol: "USDC"
)
```

### [2/4] Vault Share Token
```typescript
VaultToken.initialize(
  admin: vaultAddress, // Only vault can mint/burn
  name: "AVShare",
  symbol: "AVLT"
)
```

### [3/4] Anchor Registry
```typescript
AnchorRegistry.initialize(
  admin: deployer,
  vault_token: govTokenAddress,
  min_collateral_ratio_bps: 1000  // 10%
)
```

### [4/4] Corridor Pool Vault
```typescript
CorridorVault.initialize(
  admin: deployer,
  token: usdcAddress,
  gov_token: govTokenAddress,
  optimal_utilization: 8000,  // 80%
  base_fee_bps: 100,         // 1%
  slope_1_bps: 400,          // 4%
  slope_2_bps: 5000          // 50%
)
```

<Info>
The initialization script is **idempotent** — if a contract is already initialized, it skips silently instead of reverting.
</Info>

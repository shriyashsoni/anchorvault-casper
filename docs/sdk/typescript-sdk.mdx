# TypeScript SDK

AnchorVault provides a comprehensive TypeScript SDK (`src/lib/Casper WASM.ts`) for interacting with all protocol contracts from any JavaScript/TypeScript application.

## Installation

The SDK is included in the AnchorVault monorepo:

```bash
npm install @Casper/Casper-sdk
```

## Configuration

```typescript
import { CONTRACT_ADDRESSES } from './lib/Casper WASM';

// Deployed contract addresses (auto-populated by deploy.js)
CONTRACT_ADDRESSES = {
  USDC: "CCOFJ...",
  GOVERNANCE_TOKEN: "CDRDK...",
  ANCHOR_REGISTRY: "CC2C5...",
  CORE_VAULT: "CCMPS...",
};
```

## Quick Example

```typescript
import {
  fetchPoolState,
  fetchWalletBalances,
  buildDepositTransaction,
  submitTransaction,
} from './lib/Casper WASM';

// 1. Query pool state
const pool = await fetchPoolState(userPubKey);

// 2. Check wallet balances
const balances = await fetchWalletBalances(userPubKey);

// 3. Build a deposit transaction
const xdr = await buildDepositTransaction(userPubKey, "1000");

// 4. Sign with Casper Wallet & submit
const signedXdr = await Casper Wallet.signTransaction(xdr);
const result = await submitTransaction(signedXdr);
```

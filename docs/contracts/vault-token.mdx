# Vault Token ($AVLT) — `vault_token`

The Vault Token is an **ERC20-compatible share token** on Casper Casper WASM. It represents LP ownership in the corridor pool.

## Token Details

| Property | Value |
|:---------|:------|
| Name | AVShare |
| Symbol | AVLT |
| Decimals | 7 (Casper standard) |
| Admin | Corridor Pool Vault contract |
| Mintable | Yes (by admin only) |
| Burnable | Yes (by admin only) |

<Tip>
The $AVLT token doubles as both LP share representation AND anchor collateral. Anchors lock $AVLT in the registry, while LPs hold $AVLT to represent pool ownership.
</Tip>

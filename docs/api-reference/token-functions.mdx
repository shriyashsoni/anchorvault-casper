## `mint`

<ParamField path="to" type="Address" required>Recipient.</ParamField>
<ParamField path="amount" type="i128" required>Amount to mint.</ParamField>

Burn tokens. Admin (Vault) only.

<ParamField path="from" type="Address" required>Address to burn from.</ParamField>
<ParamField path="amount" type="i128" required>Amount to burn.</ParamField>
